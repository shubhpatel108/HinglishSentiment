git bhi kavita kee anek vidhaon men se ek vidha hai. is vidha men vyakt huee kavita gey hoti hai. isalie git kee pahachan usakee geyata men nihit hoti hai. isalie git tabhi git hai jab vah gey ho. isalie jo kavita gey nahin hai vah git nahin hai. kavita ke gey hone ka matalab hai ki usamen gan ke tattv hain aur vah gayi ja sakati hai. 

gan ke vah tattv, jo kavita ko geyata pradan karate hain, vah usakee lay men pravahit rahate hain aur lay men pravahit rahana hi unaka achook dharm aur aacharan hai. isalie bina gan ke tatvon kee lay kee kavita git nahin ho sakati. gan ke vah tatv-lay men pravahit vah tattv-git men sannadv shabdon kee vyavastha kayam kiye rahate hain. vah vyavastha svar-mukhar vyavastha hoti hai. us svar-mukhar vyavastha men vyanjanon ke uchcharanon ke roop (aakar) pratishthit rahate hain. 

uchcharanon ke inhin rupon kee sanhati lay ka nam pati hai. us sanhati men aadami kee sans kee ek nishchit lamban-chaudan aur oonchan-nichan hoti hai. us sanhati men sannaddh shabdon ke akshar-akshar ke bich ka antaral hrasv aur dirgh svaron se bhara rahata hai aur isi prakar shabd-shabd ke bich ka antaral bhi vaise hi svaron ke bharav se bhara rahata hai. yadi svaron ka bharav n ho to akshar-akshar ke bich kee doori tay karana kathin hoga. aise bharav ke abhav men shabd-shabd ke bich kee doori bhi tay karana kathin hoga. isalie geyata aur kuchh nahin, git kee gatimayata hai. 

git kee yah gatimayata jab svaron ke bharav se ek suniyojit kram-badvata pa leti hain tab sangitatmak ho jati hai. sangitatmak ho jane ka matalab hai ki git ko usaka suniyojit pravah kiya jata hai aur vah gaya ja sakata hai. koee bhi kabhi use gaye to use usi pravah se gana hoga. us pravah kee disha nirdharit disha hoti hai. 

nirdharit disha se uchcharit hokar hi git grahaniy hota hai aur grahan karane vale par apana prabhav dalata hai. lekin git matra-sangit kee abhivyakti ya ikaee nahin hota. jo git matra-sangit kee abhivyakti ya ikaee hota hai vah kavy ka git nahin hota. vah to matr dhvaniyon ka niyojit pravah hota hai. kavy ke git aur sangit ke git ka yah spasht antar kabhi bhi avahelit karane yogy nahin hai. 

kavy ke git men kathy ka hona anivary hai. yah bhi anivary hai ki vah kathy shabd aur arth kee sanshlisht aur sanyukt ikaee hokar doosaron ke lie abhivyaktishil ho jay. sangitatmakata kavy ke git kee lay ko adhik svar-mukhar banati hai aur apane sookshm amoortan se kathy ko adhik spandanashil kar deti hai. yah nahin, kathy ko kavita ke tattvon ke alava bhi sangit ka ek aur aayam sulabh ho jata hai. 

us aayam se kavita ka kathy git ke roop men dhal jata hai. aise dhalane men-git ka roop pane men-kavita men aakash tattv adhik aa jata hai aur is tattv ke samahit hone se vah, sadharan kavita se kahin adhik matra men dhvaniparak hokar door-door tak desh-kal aur manav-mastishk men vyapt ho jati hai aur apana chetan aur achetan prabhav dirghakal tak dale rahati hai. nishchay hi git kee gatimayata ka geyata ka sangitamayata ka yah tatv git ke kathy ko vyanjit karane vale shabdon ke sanghatan se nirupit aur nibaddh hota hai. 

git ke kathy ko vyanjit karane vale shabd bhi usakee gatimayata-geyata-sangitamayata ke tatv se sanghabadv hone ke lie vivash hote hain aur samarpit hote hain aur tab kahin vah sanghatan ka kritiroop pate hain. matalab yah ki git ka kathy tattv aur gey tattv ek-doosare ke niyamak aur nirdharak hote hain. unamen ek-doosare ke prati pragadh aatmiyata hoti hai. 

ek ke bina doosara ji nahin sakata, poora vikas nahin pa sakata. ek ka paryavasan doosare ka paryavasan hota hai. isalie git ke gey-tattv kee, git se alag, usake kathy se alag, vyakhya karana moolabhoot bhool hogi. vaise hi git ke kathy-tatv kee, git se alag, usakee geyata se alag, vyakhya karana moolabhoot bhool hogi. un donon tatvon ka aakalan unake sanyukt aur sanpusht roop men hi karana hoga. git apane jis roop aur arth men aaj git svikar kiya jata hai apane us roop auà¤± arth men vah pahale svikar nahin kiya jata tha. 

karan yah tha ki pahale kabhi vah sambhavanaen utpann nahin huee thin jo sambhavanaen aaj utpann ho chukee hain aur git ko apane anuroop gadh rahi hain. git bhi apane roop aur arth ke lie utana hi parivesh par nirbhar hota hai jitana aadami aur aadami ke karm aur bhav aur vichar. 

itihas kee yugin pravrittiyan aadami ke yug-bodh ko aur usakee kala-kshamata ko adhik-se-adhik prabhavit karati hain. gitakar ka vyaktitv apane yug ke itihas kee simaon men banata hai. yadi gitakar sashakt aur samarth huaa to usaka vyaktitv bhi apane yug ke itihas ko banata hai. vaise, sadharannataya, vastu-sthiti yahi rahati hai ki gitakar apane yug se prabhavit raha karata hai aur jo git vah deta hai apane yugabodh ke anuroop hi deta hai. 

git ke kathy aur shilp donon hi yugabodh ko hi rupayit kiye rahate hain. pahale, yani bahut pahale, hindi men kavita hi gayi jati thi. vah kavita, chahe matrik ho ya varnik, isaka koee asar usake gaye jane men nahin padata tha. use sangitagy gate the. sadharannajan bhi use apani kshamata ke anusar gane kee tarah gunaguna lete the. vadyayantron ke sath bhi vah gayi jati thi. 

tab is bat ka bhi dhyan nahin rakkha jata tha ki gey kavita ko varnnana-tmak ya tathyaparak n hona chahie balki use aantarik manovegon kee abhivyakti karana chahie. gayak ka kanth un kavitaon ko apane sangit se bandh leta tha. vah tal aur tan se nibaddh hokar geyata ka tattv pa leti thi. jab vah gayi nahin jati thi tab vah matr chhand rahati thi. kavitt aur savaiye bhi badi khoobi se sangitaparakata pa liya karate the. tab chhandon se alag git ka apana niji astitv n tha. yah bhi hota tha ki sangit kee raginiyon men-ragon men kuchhek kushal gayak kuchh kathy samahit kar lete the aur unhen hi sangit kee kriti ke roop men unake anuyayi shishy ga-baja lete the. 

tab bhi lok git the. parantu vah kavy ke antargat nahin aate the. kavy kee manyataen unake pratikool thin. ataev vah lok git janasadharan ke dainik jivan ke vishesh avasaron se jud kar kanth-kanth se nikalate rahe aur lokamanas ka manoranjan karate rahe. unhen tab tak sahityik pratishtha nahin mili thi. kavigan kavy ke siddhanton ke aadhar par hi kavitaon ka srijan karate rahe. vah log lok dhunen lekar kavita kee rachana karane men sanlagn nahin hue. vah kavitaen devi-devataon se sambandhit hon ya nirakar ya sakar brahm se hon athava premi eeshvar aur preyasi aatma se hon sab-kee-sab kavy kee mool manyataon ke manadand par hi aadharit hoti thin. 

"aalha" bhi dholak aur mridang ke sath ganv-ganv men gaya jata tha. vir ras ke niroopan men chhand-ke-chhand shabdon kee dhvaniyon se jhanajhana uthate the. teg-talavar kee chal aur chamak bijali ko bhi mat karane men saksham hoti thi. senaon kee umad-ghumad ko aur unake veg aur bal ko ghanaghor barasat ke badalon kee ghata ka ghamand bhi nahin laja sakata tha. vir yoddhaon kee moortiyan aisi gadhi jati thin ki svayan vir kee moorti tak tuchchh jan padane lagati thi. 

yuddh-sthal ke varnnanon men bhi aparimit kavy-kaushal ka parichay diya jata tha. vir ras ke kathy men yahi sab vyanjit hota tha. usake kathy ke chhandon ko charan kavi aur sangitagy ga-baja kar git-jaisa astitv de dete the. rasaraj shrringar ko pratishthit kiye chhand bhi apane kathy men anekanek nayak-nayikaon kee mudraon ko hi samahit kiye rahate the. nayak lobhi, lalachi, roopasakt, vasana-vilas -bhogi hote the. 

nayikaen kam-kala kee sarthakata siddh karane men apane sampoorn aangik, vachik aur manasik astitv ka khula kamuk pradarshan karati thin. chhand-ke-chhand unake deh-saundary kee yashti ban jate the. prakriti unake raman karane ke lie hi uddipit dikhayi jati thi. yah chhand bhi bade chav se sumadhur kanthon se rajadarabaron men gaye-bajaye jate the. 

in chhandon ke atirikt bhi soor, tulasi, miran aur kabir ke pad bhi gaye-bajaye jate the. vah sab ab bhi lok jivan men, ganv-ganv men, madhur svar se goonjate paye jate hain. in padon ko hi aaj ke git ka poorvaj kahana nyayasangat hoga. ve hi tab gey tatvon ko pakar hava men tairate paye jate the aur ve hi nar-nari (donon) ke kanthon se nikal kar man kee vyatha-man ka anurag-man kee dinata-aatmasamarpann-aaradhy ke prati aaradhana-aadi sookshm mano-bhavanaen vyakt kar rahe the. ve bhakt ko bhagavan se jodate the. ve din-hin ko jine kee aastha dete the. ram, rahim aur krishn unamen hi niroopit ho-kar jan-manas ke hradayeshvar ban gaye. unakee lilaon se valayit ve pad keval gey hokar hi desh-kal men goonjate rahe. 

pad vastav men, us yug ke git hain. geyata men unakee barabari chhandon men likhi aur gayi gayi kavitaen nahin kar sakeen-nahin kar sakeen. kavy kee paramparik rudhiyan padon men bhi aayin par us sima tak nahin jis sima tak ve aur kavitaon men aayin. pad fir bhi sahajata, svabhavikata aur nishchhalata se chhalakate rahe. isilie ve janata ke gale lage rahe aur aaj bhi gey bane gunagunaye jate hain. gey pad agey chhand-baddh kavitaon se koee baton men bhinn aur vishisht hote the. gey padon kee bunavat any kavitaon kee bunavat se gajhin kam aur aatmaparak adhik hoti thi. gajhin kam kee bunavat ke pad sadharannataya aadami kee sahaj svabhavik pravrittiyon ke pad hote the. unake banane vale kavy ke siddhanton ke aachary nahin hote the. 

padon ke rachayita jivan bhogate the aur apane bhoge hue jivan ko jivan kee vani men vyakti karate the. isake viparit kavy kee ritiyon ke anugami kavi jivan ko un ritiyon ke sandarbh men dekhate, sunate, samajhate aur pakadate the aur unhin ke sandarbh men use vishisht bana kar kavita ka roop dete the. is vajah se gey padon men jivan kee jhankee kuchh aur hi hoti thi aur agey chhand-baddh kavitaon men kuchh aur hi hoti thi. 

kavy kee ritiyan, kavy ke siddhant, kavy ke alankarann, kavy ke sanskar-sab kuchh aadami kee sahaj vrittiyon kee prakrit sthiti men moolabhoot parivartan kar dete the. parinam yah hota tha ki aadami kee sahaj vrittiyan vyakt hokar aadami kee sahaj bodhagamyata se door chali jati thin aur vishisht vyaktiyon kee sampreshaniyata pa leti thin. tabhi agey chhand-baddh kavitaon kee sampreshaniyata gunnagy samajikon kee ruchi ko tusht karati thi. padon kee sampreshaniyata jan-sadharan kee ruchi ko pusht karati thi. 

jab raj-darabar men kavy-goshthi hoti thi tab vani ka vaibhav-vilas raja kee rajakeey manovrittiyon kee pushti ke liye hota tha. us raj-daravar ka sampark janata ke jivan se nahin hota tha. ataev padon ka vachan aur pathan raj-darabaron men nahin hota tha. ataev padon ke nayakon ko raj-darabar ka kavi nahin banaya jata tha. janata hi-raj-daravar se bahar-un padon ke gayakon ko aashray deti thi aur usi men rah kar usi ke sukh-du:kh men vah doobe rahate the. han, mandiron ke pranganon men bhi unhen apane pad gane ka suavasar sulabh hota tha. 

jab pad-gayak kisi devasthan se sambaddh ho jata tha tab usake padon men dharm-vishesh kee chhap bhi padane lagati thi aur tab aisi sthit men usake padon se manaviy prakrit sahajata vilupt hone lagati thi. chahe raj-daravar ho-chahe devasthan-vahan pahunch kar kavi sahaj svabhavik vrittiyon ka kavi nahin rah jata tha. isilie gey padon men bhi bahut-se aise pad milenge jo dharm ke sanskaron se sanskrit ho chuke hain aur apane mool-gey svaroop se door pahunch chuke hain. 

dharmikata padon kee geyata ko dharmik sansthanon kee apani manyataon se doosare prakar kee geyata men badalati dekhi gayi hai. us badali geyata men jo manaviy pravrittiyan vyanjit hoti dekhi gayi hain vah udatt bhale hi rahi hon-daivi bhale hon-manav-manav ke jagatik sambandhon kee vyanjak hoti kadapi nahin dekhi gayin. us geyata se manav se manav judata nahin dekha gaya. us geyata ka pramukh lakshy raha hai ki vah dharm kee dhvani se, manav ko manav kee haisiyat se nahin, balki ek dharmik ko doosare dharmik se jodati rahe. yahi usane kiya bhi. us kiye ka parinam yah huaa ki gey padon ke vibhinn dharmik gey svaroop hue. 

kabhi-kabhi yah bhi huaa ki bhinn dharmavalambiyon ke gey pad ek-doosaron se takaraye aur tyajy bhi samajhe gaye. yah sab isalie huaa kyonki aadami jivan jine ke bajay dharm jine laga tha. aisi dasha men padon ke sahaj svar ka dharmik svar ho jana svabhavik tha. kuchh had tak to aise hone kee vivashata thi. us vivashata se inakar nahin kiya ja sakata. 

lekin jab dharmik pravrittiyan roodh ho jati hain aur unakee roodhivadita sahaj manav ke man ko daboch leti hain. aur us par bharapoor havi ho jati hain tab is roodhivadita se padon kee geyata men bhi bhari antar aa jata hai aur yah antar itana spasht ho jata hai ki gey pad bhi agey kavita kee tarah ke ho jate hain. vah gaye bhale hi jayen lekin unakee geyata sahaj svabhavik manaviy geyata nahin hoti. kavy ka itihas ise pramanit kar chuka hai. 

dharmik manav kee manaviyata sahaj manav kee manaviyata se alag roop aur prakar kee hoti dekhi gayi hai. tabhi to jo sahaj manaviyata miran aur soor ke padon men gey hokar vyakt huee hai vah adhik bodhagamy, sampreshaniy aur sanvedanashil hai bajay us manaviyata ke jo any dharmanuyayi kaviyon ne apane dharmik padon men gey bana kar prastut kee thi. 

matalab yah ki padon kee geyata vastav men sahaj manav ke jagatik sambandhon kee geyata hoti hai jo aage chal kar aaj ke yug ke git kee geyata men parinnat ho jati hai.
